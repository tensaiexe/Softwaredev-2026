## Step 2: Create Your Own Copy of the Template

1. Open the template repository:  
   https://github.com/alexatuea/cmp-5012b-template

2. Click **`Use this template`** (top right)

3. Select **`Create a new repository`**

4. Fill in:
   - **Owner**: your GitHub account
   - **Repository name**: e.g.
     ```
     cmp-5012b-lab1
     ```
   - **Visibility**: Public (unless told otherwise)

5. Click **`Create repository`**

‚úÖ You now have your own independent copy of the template.

1. Click the file name (e.g. `project-vision.md`)
2. Click the ‚úèÔ∏è **Edit** (pencil icon)

## Step 6: Viewing Previous Changes

GitHub automatically tracks changes.

- Click **Commits** to see:

  - When it changed
  - Who made the change
